
The Matlab function Pc3D_Hall.m calculates satellite conjunction 
probability of collision (Pc) values using the 3D-Nc approximation
method as formulated by

    Hall, D. T. "Expected Collision Rates for Tracked Satellites"
    Journal of Spacecraft and Rockets, In Press, Jan. 2021.
	
See the file PDF file included with this distribution.

Two usage/verification examples are contained in the header of the
Pc3D_Hall.m function itself, as well as in the two following functions:

	Pc3D_Hall_Example1.m
	Pc3D_Hall_Example2.m
